一口甜冰淇淋店 多頁式網站建置

【使用說明】

1. 安裝必要套件
   pip install flask

2. 啟動伺服器
   py server.py

3. 在瀏覽器打開
   http://localhost:5000

【功能說明】

- 首頁（/）
- 商品介紹頁（/products）
- 線上訂購頁（/order）
- 店鋪位置頁（/location）
- 員工管理系統（/admin）
    ➔ 輸入密碼（1234）才能查看訂單
    ➔ 錯誤會跳回首頁

【資料儲存】

- 訂單資料會存入 orders.csv
- 欄位：姓名、聯絡電話、取貨日期、便利商店類型、便利商店名稱
